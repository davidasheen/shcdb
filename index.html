Small Hydrocarbons Combustion Properties Database¶

David A. Sheen

National Institute of Standards and Technology

Last Revision Sep 26, 2017

Download this data set from GitHub

Welcome to the home page for the Small Hydrocarbons Combustion Properties Database. This is a database of laminar flame speeds and ignition delay times for a range of small hydrocarbon fuels, along with a model for simulating these experiments. The model is based on JetSurF 2.0, with the hydrogen oxidation submodel reset to its nominal values. See Sheen & Wang for more details on the base JetSurF model.
